# 🐾 宠物喂养系统使用指南

## 快速开始

### 1. 访问宠物中心
- 打开应用：http://localhost:5173/
- 点击右上角图标进入"我的奖励乐园"
- 点击"🐾 宠物中心"按钮

### 2. 选择你的宠物
从 5 种可爱宠物中选择一种：
- 🐶 **小狗** - 忠诚可爱
- 🐱 **小猫** - 温顺粘人
- 🐦 **小鸟** - 活泼好动
- 🐰 **小兔子** - 聪明伶俐
- 🐻 **小熊** - 憨厚强壮

### 3. 孵化宠物蛋
- 点击选择的宠物卡片
- 点击"开始孵化"按钮
- 宠物将从蛋开始成长

### 4. 喂养宠物
- 通过数学/英语答题获得星星⭐
- 每次喂养消耗 3 颗星星
- 每天最多喂养 3 次
- 观察成长进度条

### 5. 见证成长
宠物会经历 5 个成长阶段：
1. 🥚 **宠物蛋** (0 次喂养) - 力量值：0
2. 🐣 **幼年** (5 次喂养) - 力量值：10
3. 🐤 **青年** (15 次喂养) - 力量值：30
4. 🐕 **成年** (30 次喂养) - 力量值：60
5. 🦸 **完全体** (50 次喂养) - 力量值：100

## 功能特点

### ✨ 精美动画
- **孵化动画**：宠物蛋摇晃后破壳而出
- **升级特效**：闪光效果 + 形象变化
- **喂养动画**：爱心飘起 + 进度条增长

### 📊 成长数据
- **力量值**：显示宠物的强度
- **喂养次数**：累计喂养总次数
- **今日喂养**：显示今日剩余喂养次数
- **成长进度**：可视化进度条

### 🎯 学习激励
- 答题获得星星 → 喂养宠物 → 宠物成长
- 形成良性学习循环
- 培养孩子的学习习惯

## 技术说明

### 数据存储
- 宠物数据存储在 `localStorage`
- 键名：`userPet`
- 数据持久化，刷新页面不丢失

### 路径配置
- 使用相对路径 `./` 配置
- 支持子目录部署（如 GitHub Pages）
- Hash 模式路由

### 文件结构
```
src/
├── utils/pet/
│   └── petConfig.js       # 宠物系统配置
├── views/pet/
│   ├── PetSelect.vue      # 宠物选择页面
│   └── PetFeeding.vue     # 宠物喂养页面
└── router/
    └── index.js           # 路由配置
```

## 常见问题

### Q: 星星不够怎么办？
A: 去学习数学或英语课程，答对题目即可获得星星奖励。

### Q: 每天最多喂养几次？
A: 每天最多 3 次，防止过度消费星星。

### Q: 每次喂养需要多少星星？
A: 每次喂养消耗 3 颗星星。

### Q: 可以重新选择宠物吗？
A: 可以！在宠物中心页面底部点击"重新选择宠物"按钮，即可重新从宠物蛋开始养起。

### Q: 数据会丢失吗？
A: 数据存储在浏览器本地（localStorage），清除浏览器缓存会导致数据丢失。
