# ✅ 小熊的蜂蜜罐题目显示修复

## 问题描述

用户反馈：小熊的蜂蜜罐题目显示有问题，等号后面应该是数字，但显示的是问号。

## 问题分析

### 问题定位

**文件**：`src/components/MathQuestion.vue`

**问题代码**（第 59-61 行）：
```vue
<!-- 等号和结果 -->
<div class="equals">=</div>
<div v-if="question.type === 'fill-blank'" class="blank-slot">
  <span class="question-mark">?</span>
</div>
```

### 根本原因

在填空题（fill-blank）类型中：
- 等号后面总是显示问号 `?`
- 实际上应该显示**答案数字**（answer）
- 因为填空题是求其中一个加数，答案（总和）是已知的

### 题目逻辑

**形式 A**：`? + num2 = answer`
- 已知：num2, answer
- 求解：num1（空白位置）
- 显示：`? + 🍯... = 🍯🍯...`（答案应该显示）

**形式 B**：`num1 + ? = answer`
- 已知：num1, answer
- 求解：num2（空白位置）
- 显示：`🍯... + ? = 🍯🍯...`（答案应该显示）

## 修复方案

### 修复内容

将等号后面的问号改为显示答案的图标和数字：

```vue
<!-- 等号和结果 -->
<div class="equals">=</div>
<div v-if="question.type === 'fill-blank'" class="answer-slot">
  <div class="icon-group">
    <span 
      v-for="i in question.answer" 
      :key="'answer-' + i" 
      class="item-icon"
      :style="{ animationDelay: (question.num1 + question.num2 + i) * 0.05 + 's' }"
    >
      {{ question.icon }}
    </span>
    <span class="item-count">{{ question.answer }}</span>
  </div>
</div>
```

### 修复效果

**修复前** ❌：
```
形式 A:  🍯 + ? = ?
形式 B:  🍯 + ? = ?
```

**修复后** ✅：
```
形式 A:  🍯 + ? = 🍯🍯🍯 (显示答案数字)
形式 B:  🍯 + ? = 🍯🍯🍯 (显示答案数字)
```

## 显示示例

### 形式 A 示例
```
题目：小熊的蜂蜜罐里有一些蜂蜜，又加入了 20 罐，现在一共有 35 罐，原来有几罐呢？

显示：
🍯       +    🍯🍯... (20 个)    =    🍯🍯... (35 个)
 1                                       35
```

### 形式 B 示例
```
题目：小熊原来有 15 罐蜂蜜，现在一共有 40 罐，又加入了几罐呢？

显示：
🍯... (15 个)  +    ?    =    🍯... (40 个)
 15                                  40
```

## 技术细节

### 动画延迟计算

为了让图标逐个弹出，使用动画延迟：
```javascript
animationDelay: (question.num1 + question.num2 + i) * 0.05 + 's'
```

- 基础延迟：`question.num1 + question.num2`
- 确保答案图标在题目图标之后显示
- 每个图标间隔 0.05 秒

### 样式保持

使用 `answer-slot` 类保持原有样式：
- 橙色渐变背景
- 橙色边框
- 阴影效果
