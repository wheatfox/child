# ✅ 图标显示统一优化

## 优化说明

统一了所有大于 10 的数字的图标显示方式，严格按照"前 5 + ··· + 后 5"的格式显示。

## 优化内容

### 核心功能

**数量 ≤ 10**：显示所有图标  
**数量 > 10**：固定显示"前 5 + ··· + 后 5"

### 修改逻辑

#### 修改前
```javascript
// 前半部分：显示前 5 个
const half = Math.floor(maxDisplay / 2) // 5

// 后半部分：显示第 6-11 个（11 的时候）
const remaining = count - half
```

**问题**：11 个图标会显示 10 个（前 5 + 后 5-6 个），看起来还是太多

#### 修改后
```javascript
// 前半部分：固定显示前 5 个
for (let i = 0; i < 5; i += maxPerColumn) {
  // 显示第 1-5 个
}

// 后半部分：固定显示最后 5 个
const last5Start = count - 5
for (let i = 0; i < 5; i += maxPerColumn) {
  // 显示第 (count-4) 到 count 个
}
```

**效果**：无论数字多大，都只显示 10 个图标（前 5 + 后 5）

## 技术实现

### 算法逻辑

```javascript
const splitIntoColumns = (count, maxPerColumn = 5, maxDisplay = 10) => {
  if (count > maxDisplay) {
    const columns = []
    
    // 前半部分：固定显示前 5 个（第 1-5 个）
    for (let i = 0; i < 5; i += maxPerColumn) {
      const columnCount = Math.min(5 - i, maxPerColumn)
      columns.push({
        start: i + 1,
        count: columnCount,
        items: Array.from({ length: columnCount }, (_, j) => i + j + 1),
        type: 'normal'
      })
    }
    
    // 省略号
    columns.push({
      type: 'ellipsis'
    })
    
    // 后半部分：固定显示最后 5 个（第 count-4 到 count 个）
    const last5Start = count - 5
    for (let i = 0; i < 5; i += maxPerColumn) {
      const columnCount = Math.min(5 - i, maxPerColumn)
      const actualNum = last5Start + i + 1
      columns.push({
        start: actualNum,
        count: columnCount,
        items: Array.from({ length: columnCount }, (_, j) => last5Start + i + j + 1),
        type: 'normal'
      })
    }
    
    return columns
  }
  
  // 不超过 10，正常显示
  // ...
}
```

### 关键点

1. **前半部分**：始终显示第 1-5 个图标
2. **省略号**：表示中间省略的部分
3. **后半部分**：始终显示最后 5 个图标（第 count-4 到 count 个）
